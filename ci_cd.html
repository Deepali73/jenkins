Step 6: ‚ÄúDockerize It Like A Pro‚Äù 

sudo docker build -t myapp:latest .

Sudo zaruri hai, warna Jenkins permissions ka bawaal kar dega.

Docker Cleanup Before Rebuild

Agar tumhara Docker OS ka naam os1 hai, toh usko delete karne ke liye sabse pehle yeh command chalao:

sudo docker rm -f os1

Yeh step jaruri hai taaki naye build ke waqt conflict na ho. Ab uske baad docker image build/run commands lagao:

sudo docker build -t os1 .
sudo docker run -d --name os1 -p 5000:5000 os1

Yeh combo Jenkins ke "Execute Shell" mein add karo. Jab bhi build trigger hoga, pehle OS delete hoga, fir naya build aur run hoga ‚Äì bina error ke.
